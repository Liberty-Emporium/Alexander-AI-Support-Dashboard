Step-by-Step

Full installation guide.

Pick your agent below and follow along — no technical experience needed.

  1. 1

    Install Docker Desktop

    Agent Zero runs safely inside Docker — a lightweight virtual sandbox. Download and install it for your operating system, then start it. You'll see the Docker whale icon in your taskbar when it's running.

    Download from docker.com/products/docker-desktop → Install → Restart your computer → Open Docker Desktop and wait for the whale icon to say "Running".
    Download from docker.com/products/docker-desktop → Drag to Applications → Open Docker → Wait for whale icon in menu bar to say "Running".
    curl -fsSL https://get.docker.com | sh && sudo usermod -aG docker $USER
    Then log out and back in for the group change to take effect.
  2. 2

    Open a terminal

    Open PowerShell (Windows) or Terminal (macOS/Linux). You can search for it in your Start Menu or Spotlight.

  3. 3

    Run the installer

    Paste the command for your operating system and press Enter. This downloads and starts Agent Zero automatically.

    irm https://ps.agent-zero.ai | iex
    curl -fsSL https://bash.agent-zero.ai | bash
    curl -fsSL https://bash.agent-zero.ai | bash
  4. 4

    Open Agent Zero in your browser

    Once the installer finishes, open your browser and go to:

    http://localhost:80
    You'll see the Agent Zero web interface. From here you can configure your AI provider (OpenAI, Claude, etc.) and start giving it tasks.
  5. 5

    Add your AI API key

    In the Agent Zero settings, add your OpenAI or Anthropic API key. Don't have one? We handle this for you as part of our setup service — just book a call.

    OpenAI: platform.openai.com/api-keys
    Anthropic (Claude): console.anthropic.com/keys
  6. 6

    You're live! 🎉

    Type a task in the chat and Agent Zero will get to work. It can browse the web, write code, manage files, and run commands — all from your browser.

    Need help? Book a call with Jay and we'll get you fully configured — API keys, preferences, and your first real workflow — in under an hour.
  1. 1

    Check prerequisites

    Hermes Workspace needs Node.js 22+, git, and pnpm. Open a terminal and run this to check:

    node --version && git --version && pnpm --version
    If anything is missing: nodejs.org for Node · npm install -g pnpm for pnpm · git-scm.com for git.
  2. 2

    Run the installer

    Pick your operating system, paste the command in your terminal, and press Enter. It installs everything automatically.

    curl -fsSL https://raw.githubusercontent.com/Liberty-Emporium/Hermes-Workspace-Alexander-AI/main/install.sh | bash
    Windows requires WSL2 (Windows Subsystem for Linux). Open PowerShell as Administrator and run:
    wsl --install
    Restart your computer, then open the new Ubuntu app and run:
    curl -fsSL https://raw.githubusercontent.com/Liberty-Emporium/Hermes-Workspace-Alexander-AI/main/install.sh | bash
    This takes 2–5 minutes. Wait for "Install complete!" before moving on.
  3. 3

    Start Hermes (one command)

    Pick your OS and run the start command:

    curl -fsSL https://raw.githubusercontent.com/Liberty-Emporium/Hermes-Workspace-Alexander-AI/main/start-hermes.sh | bash
    In your Ubuntu (WSL2) terminal:
    curl -fsSL https://raw.githubusercontent.com/Liberty-Emporium/Hermes-Workspace-Alexander-AI/main/start-hermes.sh | bash

    Or start manually in two separate terminals:

    hermes gateway run
    cd ~/hermes-workspace && pnpm dev
  4. 4

    Open the workspace in your browser

    Go to:

    http://localhost:3000
    You'll see the Hermes Workspace UI. Create your first session, connect an AI provider, and start chatting!
  5. 5

    Add your AI API key

    In Settings → Providers, add your API key for OpenAI, Claude, or any other supported provider.

    OpenAI: platform.openai.com/api-keys
    Anthropic (Claude): console.anthropic.com/keys
    We handle this for you as part of our setup service.
  6. 6

    You're live! 🎉

    Hermes Workspace is running on your machine. Your data never leaves your computer. Start a chat, use multiple AI models side by side, or set up agent swarms to tackle big tasks.

    Need help? Book a call and Jay will configure everything for you — providers, memory, skills, and your first workflow — all set up and ready to go.
